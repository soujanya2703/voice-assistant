# Voice Assistant with ElevenLabs

A Python voice assistant that interacts with you in real-time using **ElevenLabs Conversational AI**. Speak naturally, and the agent responds based on your voice input.

---

## Features

- Real-time voice interaction using microphone input.
- Agent responses printed to console.
- Easy setup with `.env` for API keys.
- Built with Python 3.11+.

---

## Setup

1. **Clone the repository:**

```bash
git clone https://github.com/soujanya2703/voice-assistant.git
cd voice-assistant
pip install -r requirements.txt
cp .env.example .env
ELEVENLABS_API_KEY=your_api_key_here
AGENT_KEY=your_agent_key_here
